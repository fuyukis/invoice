# 請求書・納品書作成アプリ - Cloudflare版

サーバーレスで動作する請求書管理システムです。

## 🌟 特徴

- ✅ **完全無料で始められる** - Cloudflareの無料枠で十分
- ✅ **グローバル対応** - 世界中どこでも高速アクセス
- ✅ **データ永続化** - Cloudflare D1データベースに保存
- ✅ **マルチユーザー対応** - 複数人で使える
- ✅ **日本語・英語対応** - 国内・海外請求書に対応
- ✅ **多通貨対応** - 12種類の通貨に対応

## 📦 含まれるファイル

```
invoice-app-cloudflare/
├── index.html              # フロントエンド（メインアプリ）
├── functions/
│   └── api/[[path]].js    # バックエンドAPI（Cloudflare Workers）
├── schema.sql             # データベーススキーマ
├── wrangler.toml          # Cloudflare設定ファイル
├── DEPLOYMENT.md          # デプロイ手順（詳細版）
└── README.md              # このファイル
```

## 🛠️ 技術スタック

- **フロントエンド:** HTML + Tailwind CSS + Vanilla JavaScript
- **バックエンド:** Cloudflare Workers（サーバーレス）
- **データベース:** Cloudflare D1（SQLite互換）
- **認証:** セッションベース認証
- **ホスティング:** Cloudflare Pages
